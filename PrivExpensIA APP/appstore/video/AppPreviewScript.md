# 📹 PrivExpensIA - Script Vidéo App Store (30 secondes)

## 🎯 Objectif
Démonstration dynamique des fonctionnalités clés de PrivExpensIA en 30 secondes pour maximiser les conversions App Store.

## ⏱️ Timeline Détaillée

### **0-3s : Ouverture Impact**
- **Visuel** : Logo PrivExpensIA avec animation d'apparition
- **Texte overlay** : "PrivExpensIA" + "Scanner IA Révolutionnaire"
- **Voix-off** : "Révolutionnez vos dépenses avec l'IA"
- **Transition** : Fondu enchaîné vers l'app

### **3-8s : Démonstration Scanner**
- **Visuel** : 
  - Hands-on: utilisateur pointe son iPhone vers un reçu
  - Interface de scan en temps réel
  - Animation de reconnaissance IA (contours colorés)
  - Capture du reçu avec effet de flash
- **Texte overlay** : "95% de précision"
- **Voix-off** : "Scannez n'importe quel reçu avec 95% de précision"
- **Effet** : Zoom fluide sur le reçu pendant la capture

### **8-15s : Extraction Automatique**
- **Visuel** :
  - Transition animée vers l'écran de détails
  - Champs qui se remplissent automatiquement :
    * Montant : 45,67€
    * Date : 12/09/2024
    * Marchand : Carrefour Market
    * Catégorie : Alimentation (auto-sélectionnée)
  - Animation de particules IA autour des données extraites
- **Texte overlay** : "Extraction 100% automatique"
- **Voix-off** : "L'IA extrait automatiquement toutes les informations"

### **15-22s : Dashboard & Analytics**
- **Visuel** :
  - Swipe fluide vers le dashboard principal
  - Graphiques animés qui se dessinent :
    * Graphique en donut des catégories
    * Courbe des dépenses mensuelles
    * Cartes statistiques avec compteurs animés
  - Effet parallax sur les éléments
- **Texte overlay** : "Analyses intelligentes"
- **Voix-off** : "Analysez vos habitudes de consommation"

### **22-27s : Confidentialité & Export**
- **Visuel** :
  - Split-screen :
    * Gauche : Icône cadenas + "Données locales" + check marks
    * Droite : Export PDF en cours + preview du rapport
  - Animation de protection (bouclier qui apparaît)
- **Texte overlay** : "100% Privé • Export Pro"
- **Voix-off** : "Vos données restent privées. Export professionnel inclus."

### **27-30s : Call-to-Action Final**
- **Visuel** :
  - Logo PrivExpensIA centré
  - Animation d'étoiles/reviews (5 étoiles qui apparaissent)
  - Bouton "Télécharger" pulsant
- **Texte overlay** : "PrivExpensIA - Disponible maintenant"
- **Voix-off** : "Téléchargez PrivExpensIA aujourd'hui !"

## 🎨 Spécifications Techniques

### **Résolution & Format**
- **Format de sortie** : MP4 (H.264)
- **Résolution** : 1920x1080 (Full HD)
- **Frame Rate** : 30 FPS
- **Ratio d'aspect** : 16:9
- **Orientation** : Landscape avec device mockups

### **Qualité**
- **Bitrate vidéo** : 8-10 Mbps
- **Bitrate audio** : 256 kbps AAC
- **Profil colorimétrique** : sRGB
- **Compression** : H.264 High Profile

### **Contraintes App Store**
- **Durée max** : 30 secondes exactement
- **Taille max** : 500 MB
- **Langues** : Version française (principale) + sous-titres anglais
- **Safe zones** : Respecter les marges pour tous les devices

## 🎬 Transitions & Animations

### **Types de Transitions**
- **Fondu enchaîné** : Entre logo et app (0-3s)
- **Slide + parallax** : Entre écrans app (8s, 15s)
- **Zoom sémantique** : Focus sur éléments importants
- **Morphing** : Transformation de données (extraction)

### **Micro-animations**
- **Loading spinners** IA processing
- **Bounce effects** sur boutons et CTA
- **Particle systems** pour effets IA
- **Number counters** pour statistiques
- **Progress bars** pour exports

## 🎯 Métriques de Succès
