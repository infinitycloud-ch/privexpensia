# PrivExpensIA - User Guide / Guide Utilisateur / Benutzerhandbuch

---

## 🇬🇧 English

### Getting Started
1. **Scan Receipt**: Tap the camera icon to scan your receipt
2. **Review Data**: PrivExpensIA automatically extracts amount, date, and merchant
3. **Select Category**: Choose from predefined expense categories
4. **Submit**: Save your expense for approval

### Key Features
- **Smart OCR**: Automatic receipt data extraction
- **Multi-Currency**: Support for CHF, EUR, USD, JPY, and more
- **VAT Calculation**: Automatic VAT detection and calculation
- **Quick Templates**: Fast expense entry for common purchases

### Tips
- Keep receipts flat and well-lit for best scanning results
- Review extracted data before submitting
- Use quick templates for recurring expenses

## 🇯🇵 日本語

### はじめに
1. **領収書をスキャン**: カメラアイコンをタップしてスキャン
2. **データを確認**: PrivExpensIAが金額、日付、店舗を自動抽出
3. **カテゴリー選択**: 事前定義された経費カテゴリーから選択
4. **送信**: 承認のため経費を保存

### 主な機能
- **スマートOCR**: 領収書データの自動抽出
- **多通貨対応**: CHF、EUR、USD、JPY等をサポート
- **消費税計算**: 自動消費税検出と計算
- **クイックテンプレート**: よくある購入の素早い入力

### ヒント
- 最良の結果のため領収書は平らで明るい場所で
- 送信前に抽出データを確認
- 定期的な経費にはクイックテンプレートを使用
