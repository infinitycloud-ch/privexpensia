## 📸 **2. Scanner un Reçu : Guide Étape par Étape**

### Préparation du Reçu
Pour obtenir les meilleurs résultats :

✅ **À FAIRE :**
- Aplatir complètement le reçu (pas de plis)
- Nettoyer la surface s'il y a des taches
- Vérifier que tous les éléments sont visibles
- Scanner dans les 48h si possible (meilleure qualité papier)

❌ **À ÉVITER :**
- Reçus froissés ou pliés
- Papier brillant avec reflets
- Écriture manuscrite par-dessus le texte imprimé
- Scanner sous lumière directe du soleil

### Process de Scan

1. **📷 Lancer la Caméra**
   - Appuyez sur l'icône caméra en bas de l'écran
   - Positionnez le reçu dans le cadre
   - Le cadre devient vert quand la qualité est optimale

2. **🎯 Cadrage Parfait**
   - Gardez une distance de 20-30 cm du reçu
   - Assurez-vous que tout le reçu est dans le cadre
   - Les bords du reçu doivent être parallèles aux bords de l'écran
   - Attendez que l'autofocus se stabilise

3. **💡 Éclairage Optimal**
   - **Idéal** : Lumière naturelle indirecte (près d'une fenêtre)
   - **Acceptable** : Éclairage LED blanc uniforme
   - **À éviter** : Flash direct, néons vacillants, lampes jaunes

4. **📐 Angle et Position**
   - Tenez le téléphone perpendiculaire au reçu (90°)
   - Évitez les angles obliques
   - Utilisez les deux mains pour plus de stabilité
   - Activez le minuteur si vous tremblez

### Cas Particuliers

**🧾 Reçus Longs**
- Utilisez la fonction "Mode Panorama"
- Scannez en plusieurs sections si nécessaire
- L'IA reconstituera automatiquement

**📄 Reçus Multiples**
- Scannez chaque reçu individuellement
- Utilisez la fonction "Lot" pour les regrouper
- Maximum recommandé : 5 reçus par lot

**🖨️ Reçus Thermiques Décolorés**
- Augmentez le contraste dans les paramètres
- Utilisez un fond noir derrière le reçu
- Scanner immédiatement après achat si possible

## 💰 **3. Guide TVA par Pays**

### 🇨🇭 **Suisse (CHF)**
La Suisse a un système TVA à 3 taux :

- **📊 TVA Standard : 8,1%**
  - Restaurants et hôtels
  - Transport, taxi
  - Carburant, parking
  - Fournitures de bureau
  - Matériel informatique
  - Divertissements

- **🥘 TVA Réduite : 2,5%**
  - Produits alimentaires de base
  - Boissons non-alcoolisées
  - Livres et journaux
  - Médicaments

- **🏨 TVA Hébergement : 3,7%**
  - Nuitées d'hôtel uniquement
  - Bed & Breakfast
  - Auberges de jeunesse

> 💡 **Astuce Suisse** : Les repas à l'emporter sont souvent à 2,5% au lieu de 8,1%

### 🇩🇪 **Allemagne (EUR)**
Système à 2 taux avec distinction importante :

- **📊 TVA Standard : 19%**
  - Restaurants (consommation sur place)
  - Transport longue distance
  - Carburant et parking
  - Matériel et logiciels
  - Divertissements

- **🥘 TVA Réduite : 7%**
  - Produits alimentaires
  - Transport local et taxis
  - Hôtels et hébergement
  - Livres et journaux
  - **Restaurants (à emporter)**

> 💡 **Astuce Allemagne** : Un même restaurant applique 19% pour manger sur place, 7% pour emporter !

### 🇫🇷 **France (EUR)**
Système complexe à 4 taux :

- **📊 TVA Standard : 20%**
  - Carburant et parking
  - Fournitures de bureau
  - Matériel informatique
  - Divertissements (sauf culture)

- **🏨 TVA Intermédiaire : 10%**
  - Restaurants (consommation sur place)
  - Hôtels et hébergement
  - Transport (taxi, train)

- **🥘 TVA Réduite : 5,5%**
  - Produits alimentaires
  - Livres et journaux
  - **Restaurants (à emporter)**
  - Certains événements culturels

- **📰 TVA Super-Réduite : 2,1%**
  - Médicaments remboursés
  - Presse quotidienne

### 🇯🇵 **Japon (JPY)**
Système simple à 2 taux :

- **📊 Taxe de Consommation Standard : 10%**
  - Restaurants (consommation sur place)
  - Hôtels et transport
  - Tous biens et services sauf exceptions

- **🥘 Taxe Réduite : 8%**
  - Produits alimentaires (sauf alcool)
  - **Restaurants (à emporter)**
  - Boissons non-alcoolisées

> 💡 **Particularité Japon** : La distinction emporter/sur place est très stricte

### 🇺🇸 **États-Unis (USD)**
Système décentralisé complexe :

**États sans Taxe de Vente :**
- Oregon, Montana, New Hampshire, Delaware, Alaska (partiel)

**États avec Taxe Élevée :**
- 🌟 **Californie** : 7,25% base + taxes locales (total ~8,85%)
- 🗽 **New York** : 4% base + jusqu'à 4,5% à NYC (total ~8,52%)
- 🏖️ **Washington** : 6,5% base + taxes locales (total ~9,29%)

**États avec Taxe Modérée :**
- 🤠 **Texas** : 6,25% base + jusqu'à 2% local (total ~8,20%)
- 🌴 **Floride** : 6% base + taxes locales (total ~7,01%)

> 💡 **Astuce USA** : L'app détecte automatiquement l'État via géolocalisation

## 🔧 **4. Résoudre les Problèmes Courants**

### 📷 **Photos Floues**

**Causes et Solutions :**
- ❌ **Tremblement** → ✅ Utilisez le minuteur (2-3 secondes)
- ❌ **Distance trop proche** → ✅ Reculez à 20-30 cm minimum  
- ❌ **Autofocus incorrect** → ✅ Tapez sur le reçu à l'écran
- ❌ **Vitesse trop rapide** → ✅ Attendez la stabilisation (cadre vert)

**Mode Assistance Flou :**
1. Activez "Mode Haute Résolution" dans Paramètres
2. Utilisez un support ou appuyez-vous contre un mur
3. Respirez calmement et bloquez à l'expiration

### 💡 **Mauvais Éclairage**

**Solutions selon la Situation :**

**🌅 Matin/Soir :**
- Placez-vous près d'une fenêtre
- Ajoutez une lampe LED blanche
- Évitez les ombres portées

**🌙 Nuit :**
- **Ne jamais** utiliser le flash direct
- Utilisez 2 sources lumineuses opposées
- Inclinez légèrement le reçu pour éviter les reflets

**🏢 Bureau :**
- Éteignez les néons vacillants
- Utilisez la lampe de bureau LED
- Créez un "studio" avec une feuille blanche comme fond

### 📄 **Reçus Décolorés/Anciens**

**Reçus Thermiques Pâlis :**
1. **Méthode du Froid** : Placez 5 minutes au congélateur, le texte redevient temporairement visible
2. **Contraste Maximum** : Augmentez le contraste dans l'app avant de scanner
3. **Fond Noir** : Placez une feuille noire derrière le reçu
4. **Éclairage Rasant** : Éclairez depuis le côté pour révéler les reliefs

**Reçus Tachés/Endommagés :**
1. **Nettoyage Doux** : Chiffon légèrement humide
2. **Scan Multiple** : Scannez sous différents angles
3. **Mode Manuel** : Saisissez les données manuellement si nécessaire

### ✏️ **Écritures Manuscrites**

**Gestion des Annotations :**
- L'IA ignore généralement l'écriture manuscrite
- **Tip Personnel** : Écrivez sur le côté, pas sur les montants
- **Notes de Frais** : Utilisez le champ "Notes" de l'app plutôt que le reçu
- **Mode Mixte** : Validez les données automatiques, ajoutez manuellement le reste

### 💱 **Devises Multiples**

**Reçus en Devise Étrangère :**
1. L'app détecte automatiquement la devise
2. Taux de change mis à jour quotidiennement
3. **Conversion Manuelle** : Si taux spécifique nécessaire
4. **Mode Entreprise** : Verrouillez les taux selon politique interne

## ⚡ **5. Optimiser Votre Utilisation**

### 📦 **Scan par Lots (Batch Scanning)**

**Préparer un Lot :**
1. Triez vos reçus par date ou catégorie
2. Maximum 10 reçus par session pour performance optimale
3. Scannez d'abord les plus récents (meilleure qualité)

**Technique de Production :**
- **Setup** : Préparez votre "poste" (éclairage + support)
- **Rythme** : 1 reçu toutes les 30 secondes maximum
- **Validation** : Vérifiez chaque scan avant le suivant
- **Pause** : Arrêt toutes les 5 scans pour éviter les erreurs

### 🏷️ **Gestion Intelligente des Catégories**

**Catégories Prédéfinies :**
- 🍽️ **Alimentation** (vert) - TVA réduite souvent

- 🚗 **Transport** (bleu) - Includes taxi, train, avion
- ⛽ **Carburant** (rouge) - TVA standard généralement
- 🏢 **Bureau** (gris) - Fournitures, matériel
- 🎯 **Divertissement** (rose) - Attention aux taux culturels

- 💻 **Logiciel** (indigo) - Vérifier taxation numérique

**Personnalisation Avancée :**
