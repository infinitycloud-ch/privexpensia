# PrivExpensIA - TestFlight Beta Information

## Beta App Description
PrivExpensIA is an intelligent expense tracking app that uses advanced AI and OCR technology to automatically extract information from receipts. All processing happens on your device, ensuring 100% privacy.

## What to Test

### 1. Receipt Scanning
- Test camera scanning with different receipt types
- Try photo library import
- Verify OCR accuracy
- Check multi-language support (8 languages)

### 2. AI Extraction
- Test automatic merchant detection
- Verify amount and tax extraction
- Check category assignment
- Test item detection

### 3. User Interface
- Navigate through all screens
- Test glass morphism effects
- Check animations performance
- Verify dark/light mode

### 4. Data Management
- Create and edit expenses
- Delete expenses
- Export data (PDF/CSV)
- Check statistics accuracy

### 5. Performance
- Monitor app responsiveness
- Check memory usage
- Test with large datasets
- Verify offline functionality
