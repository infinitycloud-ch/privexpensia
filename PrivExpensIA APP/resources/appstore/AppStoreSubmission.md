### Review Notes for Apple

#### For App Review Team

**Important Implementation Notes**:

1. **Offline-First Architecture**: 
   - PrivExpensIA works completely offline
   - No network permissions required
   - All data stored locally using Core Data
   - No user accounts or registration needed

2. **Camera Usage**:
   - Camera is used exclusively for receipt scanning
   - OCR processing happens entirely on-device
   - No images are transmitted or stored permanently
   - Camera permission requested only when user accesses scanner

3. **Privacy Implementation**:
   - Zero data collection or transmission
   - No analytics frameworks integrated
   - No third-party SDKs that transmit data
   - Complete GDPR compliance by design

4. **AI/ML Features**:
   - OCR uses Apple's Vision framework (on-device)
   - Machine learning models run locally
   - No cloud-based AI services used
   - Models are part of app bundle

5. **Testing Instructions**:
   - Launch app → automatic onboarding
   - Grant camera permission when prompted
   - Use "Demo Mode" button to explore with sample data
   - Scanner tab → point at any receipt or document
   - All features accessible without setup

#### Known iOS Version Compatibility
- **Minimum**: iOS 15.0
- **Optimized for**: iOS 17.0+
- **Tested on**: All supported devices from iPhone SE 2nd gen to iPhone 15 Pro Max
- **iPad**: Full compatibility with adapted interface

#### Performance Characteristics
- **Launch time**: < 2 seconds on all supported devices

- **Memory usage**: 45-120MB typical, 200MB maximum
