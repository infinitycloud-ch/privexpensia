## Version 1.0.0 (Build 1) - Initial Beta
*Release Date: September 11, 2025*

### ✨ Core Features
- **Smart Receipt Scanning**: Advanced OCR with 8-language support
- **AI-Powered Extraction**: Automatic merchant, amount, and category detection
- **100% Privacy**: All processing on-device, no data leaves your phone
- **Glass UI Design**: Beautiful frosted glass interface
- **Core Data Storage**: Secure local database
- **Statistics & Charts**: Visual spending insights
- **Export Options**: PDF and CSV export support

### 🚀 Key Technologies
- Vision Framework for OCR
- MLX Framework for AI
- Qwen2.5-0.5B model (4-bit quantized)
- SwiftUI with glass morphism
- Core Data persistence
- Charts framework

### 📊 Performance Targets
- OCR processing: < 2 seconds
- AI inference: < 300ms
- Memory usage: < 150MB
- 60 FPS animations
- Zero network requests

- No account required
- No internet connection needed
- No data collection
- No third-party services
- Local processing only
