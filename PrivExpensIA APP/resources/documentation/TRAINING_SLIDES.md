**A:** PrivExpensIA is designed offline-first. All processing occurs on-device with no cloud dependency. Receipts are processed normally and sync when connectivity returns. Zero functionality is lost offline.

**A:** The Qwen2.5 model runs entirely within iOS's secure enclave. Data never leaves the device, eliminating cloud security risks. The model itself is encrypted and cannot be extracted or reverse-engineered.
